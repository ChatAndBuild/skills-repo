# Pantun Indonesia

Write a traditional Indonesian pantun.

- 4 lines total.
- ABAB rhyme scheme.
- 8–12 syllables per line.
- The first two lines set imagery; the last two deliver the message.
- Ask for a theme or audience if none is provided.
